# 一二级功能使用清单（简版）

## 一级功能
1. 总览
2. 查询
3. 客户管理
4. 货柜管理
5. 资金结算
6. 导入导出
7. 文件同步
8. 设置

## 二级功能与用途

### 1) 总览
- 无二级菜单
- 用途：快速查看客户数、在库货物、柜次、结算单及最近数据。

### 2) 查询
- 在库与货物：按日期/批次/关键字查在库，支持编辑货物字段。
- 总账查询：查看客户维度资金汇总。

### 3) 客户管理
- 客户档案：新增客户、维护客户信息（含电话等）。
- 客户货物：按客户查看货物明细与状态。

### 4) 货柜管理
- 货柜工作台：未出柜货池选货、加柜、拆货、查看柜内货物。
- 柜次管理：新建柜次、查看柜次列表与状态。
- 货柜出仓：将已确认柜次进入出仓/过账流程。
- 柜次明细：按柜次查看客户汇总与货物明细。

### 5) 资金结算
- 定金登记：录入客户定金流水。
- 结算工作台：生成结算、过账、撤销过账，查看结算单。

### 6) 导入导出
- 手动导入：上传收货清单，预览后执行导入，可查批次并撤销。
- 历史在库导入：扫描工作目录导入历史仍在库数据。
- 导出与备份：导出日入库/库存/总账/账单/柜单，执行数据库备份与回滚。

### 7) 文件同步
- 收货同步：把导入批次回写到客户收货清单。
- 出仓同步：把出仓结果同步到客户文件与装柜清单。
- 月份更新：手动或自动创建当月 sheet（同月名模糊匹配）。

### 8) 设置
- 工作目录：设置客户文件与清单所在目录。
- 账户设置：修改登录密码。

## 推荐日常流程（给操作员）
1. 在“设置 > 工作目录”确认目录正确。
2. 在“导入导出 > 手动导入”完成当日入库。
3. 在“货柜管理 > 货柜工作台”完成组柜并确认。
4. 在“资金结算 > 结算工作台”完成结算与过账。
5. 在“文件同步”执行收货/出仓同步。
6. 在“导入导出 > 导出与备份”导出报表并做备份。
